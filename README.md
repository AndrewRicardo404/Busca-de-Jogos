# 🎮 Busca de Jogos

Aplicação web que busca jogos usando o API CheapShark.

## Como Funciona?
- Buscar jogos
- Exibir em tabela
- Salvar no localStorage
- Remover itens

## Códigos Usados
- HTML
- CSS
- JavaScript Vanilla

## Como executar
Abra o index.html no navegador e Digite pelo nome de um Jogo.
