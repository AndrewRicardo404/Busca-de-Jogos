# 🎮 Busca de Jogos

Aplicação web que busca jogos usando a API CheapShark.

## 🚀 Funcionalidades
- Buscar jogos
- Exibir resultados em tabela
- Mostrar preços, nomes e imagens
- Remover itens

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript
